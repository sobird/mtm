# Mdx Playground

## MDX 是什么？
MDX 赋予你在 markdown 内容当中使用 JSX 的能力。
你可以导入（import）如见，例如交互式图标或警告框，并将它们嵌入 到你所书写的内容当中。
这让利用组件来编写较长的内容成为了一场革命。 🚀
实际上，MDX 可以看作是一种融合了 markdown 和 JSX 的格式，就像下面的示例：

<div style={{backgroundColor: 'violet', padding: '1rem'}}>
  尝试将背景颜色更改为 `tomato`.
</div>

  // 打印props
  console.log('props', props)
}
