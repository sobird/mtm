## FAQ

### [qiankun] lifecycle not found from xxx entry exports, fallback to get from window['xxx']

qiankun是通过html中的最后一个script文件，获取qiankun的生命周期，所以如果最后一个不是入口js文件就拿不到生命周期。

可在webpack.config.js文件中修改下面配置修复此问题
```js
const config = {
  ...
  plugins: [
    new HtmlPlugin({
      ...
      chunks: [ 'app' ]
    }),
  ]
  ...
}
```
